# pip教程

## 1. pip常用命令

```text
    查看已安装包                pip list
    安装包                      pip install package
        指定版本                pip install package==version
                               pip install 'package>=version'
    更新包                      pip install --upgrade package
    删除包                      pip uninstall package
    显示包信息                  pip show
    导出依赖包                  pip freeze > requirements.txt
    安装依赖包                  pip install -r requirements.txt
```
